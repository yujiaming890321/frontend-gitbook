Package Env

packjson 环境变量常用库

cross-env

配置环境变量

{
 "scripts": {
   "build": "cross-env NODE_ENV=production webpack --config build/webpack.config.js"
 }
}

dotenv

零依赖模块，从.env 加载配置文件

"build:dev": "dotenv -e .env.dev -e .env react-app-rewired build",
"build:pre": "dotenv -e .env.pre -e .env react-app-rewired build",

husky

执行 git hooks， v4 与 v7 差异较大，4.3.8 是 packagjson 配置， v7 是文件配件。

f2elint

F2ELint 是《阿里巴巴前端规约》的配套 Lint 工具，包括 Linter 依赖，如 ESLint、stylelint、commitlint、markdownlint 等

lint-staged

防止错误代码提交

"husky": {
    "hooks": {
        "pre-commit": "lint-staged",
        "commit-msg": "commitlint -E HUSKY_GIT_PARAMS"
    }
},
"lint-staged": {
    "*.{js,css,md,ts,tsx}": [
        "prettier --write",
        "git add"
    ]
}

commitlint

检查提交 msg 是否合规

// commitlint.config.js
module.exports = {
    extends: ['@commitlint/config-conventional'],
    rules: {
        'type-enum': [
            2,
            'always',
            [
                'feat',
                'fix',
                'docs',
                'style',
                'refactor',
                'test',
                'chore',
                'comment',
                'build',
                'merge',
            ],
        ],
        'subject-full-stop': [0, 'never'],
        'subject-case': [0, 'never'],
    },
}
